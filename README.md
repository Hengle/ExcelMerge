# ExcelMerge

[![Build status](https://ci.appveyor.com/api/projects/status/ven9wb4k5wrrajub?svg=true)](https://ci.appveyor.com/project/luxuia/excelmerge)

Features:

    自带svn日志，方便看各个版本的差异
    预先对excel的行列按内容排序，忽略行位置变化导致的修改
    可以查看已打开的文档
    默认找到第一个有差异的sheet
    背景色表示差异类型，黄色[修改]，灰色[删除]，绿色[新增]
